## Uso

Para ejecutar el local :

-   npm i
-   ionic serve

Para ejecutar en Android studio

-   ionic build
-   ionic cap add (android / ios)
-   ionic cap open (android / ios)

Para ejecutar en Android studio hot reload

-   ionic capacitor run android -l --external
